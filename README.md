# TourOfHerosWebApp

This is a web application that allows users to explore a list of heroes, search for specific heroes, and view detailed information about each hero. The application is built using Angular and communicates with a backend ASP.NET server to fetch and display data.
<https://www.thecsharpacademy.com/project/31/tour-of-heroes>

## Features

- Uses Angular CLI to generate and manage the project.
- Implements a search functionality to find heroes by name.
- Displays a list of heroes with options to view detailed information.
- Provides a user-friendly interface for navigating through the list of heroes.
- Uses HTTP client module to fetch data from the backend server.
- Includes unit tests to ensure the functionality of the application.
- Supports HTTPS for secure communication with the backend server.
- Configures a proxy to route API calls to the backend server.

## Getting Started

To run the application, follow these steps:

1. Clone the repository to your local machine.
2. Open the solution in Visual Studio.
3. Build the solution to restore npm packages and compile the code.
4. Run the `tourofHeroeswebapp.client` project to start the web application.

## Dependencies

- Angular CLI: The application uses this tool to generate and manage the Angular project.
- HttpClientModule: The application uses this module to make HTTP requests to the backend server.
- Karma: The application uses this tool to run unit tests.
- Jest: The application uses this tool for additional testing support.

## Usage

1. The application will display a list of heroes with options to view detailed information.
2. Use the search box to find heroes by name.
3. Click on a hero to view detailed information about the hero.
